# RPGCC (🇬🇧)
A project that consists on a desktop app, developed using Java and Swing to simulate a DnD-like character creation system. Made using NetBeans IDE as a task to practice how event-based GUIs work.
<br/>
<br/>
In order to execute this program, you'll need an IDE that's compatible with Java (just like NetBeans! And in case you have a Windows device, you will need to have Cygwin installed in your device too), and paste in its terminal the following code:

```
git clone https://github.com/rin-5/RPGCC.git
```
And, you will be able to execute the main class and let the program run its course! <br />
# RPGCC (🇪🇸)
Un proyecto que consiste en una app de escritorio desarrollada usando Java y Swing. Simula una aplicación de creación de personajes al estilo de Mazmorras y Dragones. Esta tarea se hizo usando el IDE Netbeans, cuyo objetivo era practicar cómo funcionan las GUIs basadas en eventos.
<br />
<br />
Para copiarlo en un IDE como Netbeans (en cuyo caso tendrás que tener instrlado Cygwin en el caso de que el dispositivo sea Windows), debes pegar en la terminal el siguiente código:
```
git clone https://github.com/rin-5/RPGCC.git
```
Tras eso, solo hay que ejecutar el programa e inmediatamente la clase principal se ejecutará.
